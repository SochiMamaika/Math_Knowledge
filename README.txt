Когда у нас создается окно у нас запускается таймер который каждую секунду вызывает метод OnTimeout где считаются секунды, при нажатии кнопки запускается метод OnNotify
в котором переменная отвечающее за пройденное количество секунд сбрасывается, также там сверяется правильность ответа
(Если правильно идем дальше в метод Ask составлять следующее задание) (Если неправильно то запускается метод Lose).

далее запускаем файл gui командой (./gui)

"-" - Включить отрицательный флаг, чтобы начать ввод отрицательного числа (выключать не надо))
"q" - Закрыть приложение
