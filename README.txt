Backspace - Сбросить введенное
"-" - Включить отрицательный флаг, чтобы начать ввод отрицательного числа (выключать не надо))
"q" - Закрыть приложение
Кнопка реагирует как на нажатие ЛКМ так и на Enter
Компилировать программу командой (g++ window.cc GUI.cc Math_Knowledge.cc button.cc text.cc image.cc  -o gui `pkg-config --cflags --libs gtk+-3.0`)
далее запускаем файл gui командой (./gui)
